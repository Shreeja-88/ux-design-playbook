# Stephen Wendel's CREATE Action Funnel

## What is CREATE Action Funnel?

The **CREATE Action Funnel** is a behavioral design framework created by Stephen Wendel.

It helps designers and product teams understand:

* Why users take action
* Where users drop off
* How to improve user behavior in products

The framework is mainly used in:

* UX/UI Design
* Product Design
* Behavioral Science
* Habit-forming applications

---

# CREATE Framework

```text id="a2lybf"
C → Cue
R → Reaction
E → Evaluation
A → Ability
T → Timing
E → Execution
```

---

# Stages of CREATE Funnel

## 1. Cue

A trigger or reminder that tells the user to act.

### Examples

* Push notification
* Email reminder
* Button highlight
* Alarm

### Goal

Make users notice the action.

---

## 2. Reaction

The user’s immediate emotional response to the cue.

### Positive Reaction

* “This looks useful!”
* “I should check this.”

### Negative Reaction

* “Too much work.”
* “Not interested.”

### Goal

Create curiosity or excitement.

---

## 3. Evaluation

The user thinks about:

* Benefits
* Effort
* Time required

Users decide whether the action is worth doing.

### Example

“Is this worth my time?”

### Goal

Increase perceived value and reduce effort.

---

## 4. Ability

The user must feel capable of doing the task.

### Designers should:

* Simplify tasks
* Reduce steps
* Provide guidance

### Example

One-click signup improves ability.

---

## 5. Timing

The action must happen at the correct moment.

### Good Timing

* Food delivery ads during lunch
* Study reminders before exams

### Bad Timing

* Notifications during meetings

### Goal

Prompt users when they are most likely to act.

---

## 6. Execution

The user successfully completes the action.

### Example

* Completing a purchase
* Posting a photo
* Submitting a form

### Goal

Ensure smooth and successful completion.

---

# Simple Example – Instagram Notification

| Stage      | Example                  |
| ---------- | ------------------------ |
| Cue        | Notification appears     |
| Reaction   | “Someone liked my post!” |
| Evaluation | “Let me check quickly.”  |
| Ability    | Easy to open app         |
| Timing     | User is free             |
| Execution  | User opens Instagram     |

---

# Importance in UX/UI Design

CREATE helps designers:

* Understand user behavior
* Reduce user drop-offs
* Improve engagement
* Build habit-forming products
* Optimize user journeys

---

# Advantages

* Easy behavioral framework
* Helps improve conversions
* Useful for product strategy
* Identifies user pain points

---

# Limitations

* User motivation changes over time
* Different cultures react differently
* Timing may vary for users

---

# Key Takeaways

* Users must pass through all CREATE stages to complete an action.
* If users drop off, designers should identify which stage failed.
* Simpler actions improve success rates.
* Good timing and prompts increase engagement.

---

# Quick Summary

```text id="71q8gj"
CREATE Funnel:

Cue → Notice action
Reaction → Emotional response
Evaluation → Is it worth it?
Ability → Can I do it?
Timing → Is this the right moment?
Execution → Complete the behavior
```
