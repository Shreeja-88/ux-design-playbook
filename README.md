# ux-design-playbook

This repository contains my learning notes, frameworks, resources, and practical understanding of UX Design, Behavioral Psychology, Persuasive Technology, and Product Thinking.

## Topics Covered
- Human Decision Making
- Behavioral Economics
- Nudge Theory
- BJ Fogg Behavior Model
- Hook Model
- UX Patterns & Best Practices
- Wireframing & Prototyping
- Business Models
- User Personas
- Gamification
- A/B Testing
- Design Systems
- Figma & UX Tools

## Goal
To become better at designing products that are intuitive, engaging, and behavior-driven.
